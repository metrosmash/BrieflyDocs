## What is BrieflyDocs?

BrieflyDocs is a **Document Intelligence Agent** that transforms any uploaded document вҖ” PDF, image, or plain text вҖ” into structured, actionable metadata in seconds. No more manually skimming 30-page reports. Drop it in, get the signal, move on.

Under the hood it uses a large-context AI model (no chunking, no RAG) to read the entire document at once, preserving nuance that traditional extraction pipelines miss.

## The Prompt

BrieflyDocs is powered by a carefully engineered system prompt that you can find and customize in `src/lib/gemini.js`. Key design decisions:

- **No chunking instruction** вҖ” explicitly tells the model to analyze the full document in one pass
- **Strict JSON schema** вҖ” returns raw JSON with no markdown fences or prose
- **Typed entity extraction** вҖ” classifies each entity as Person, Organization, or Date
- **Confidence scoring** вҖ” lower scores surface when the document is ambiguous or multi-domain
- **Visual document fallback** вҖ” for infographics or image-only PDFs, the model describes visual data in the summary field
